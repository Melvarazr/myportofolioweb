BAS Hiring: Recruitment Information System

Project Description

BAS Hiring is an integrated recruitment management system designed to streamline and optimize the hiring process for PT Barokah Amanah Sentosa. This system automates recruitment workflows, including account management, candidate application processing, job posting, and outsourcing service requests. The platform enables recruiters, general managers, and candidates to interact efficiently through various dashboards, notifications, and real-time updates.

Key Features

Account Management

Admin can manage user accounts, ensuring only valid users access the system.

Job Application

Candidates can apply for jobs through a seamless registration and application process.

Dashboard

Custom dashboards for General Managers, Admins, and Recruiters to track data in real-time.

Notifications

Real-time notifications for candidates and vendors about their application or request status.

Job Postings

Recruiters can publish and manage job listings and job applications.

Outsourcing Services

Integration with vendors to manage outsourcing requests.
